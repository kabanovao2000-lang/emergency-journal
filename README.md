 Электронный журнал учёта аварийных ситуаций

 О проекте

Система для автоматизации учёта и обработки аварийных ситуаций на предприятии.

 Функциональность

- Регистрация аварийных ситуаций
- Назначение ответственных сотрудников
- Контроль выполнения работ
- Формирование отчётов и статистики

 Технологический стек

| Компонент | Технология |
|-----------|------------|
| Backend | Python + Django |
| Frontend | React |
| База данных | PostgreSQL |
| Документация | Swagger/OpenAPI |
